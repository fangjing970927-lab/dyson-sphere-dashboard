## ✅ 步骤1：启用GitHub Pages

**请在浏览器中完成以下操作：**

1. **访问您的仓库**
   ```
   https://github.com/fangjing970927-lab/dyson-sphere-dashboard
   ```

2. **进入设置页面**
   - 点击仓库页面上方的 **"Settings"** 标签

3. **找到GitHub Pages设置**
   - 在左侧菜单中，滚动找到 **"Pages"** 选项
   - 点击 **"Pages"**

4. **配置部署源**
   - **Source**: 选择 **"Deploy from a branch"**
   - **Branch**: 选择 **"main"**
   - **Directory**: 选择 **"/ (root)"**
   - 点击 **"Save"** 按钮

5. **等待部署**
   - GitHub会显示："Your site is ready to be published at..."
   - 等待1-2分钟让GitHub完成部署
